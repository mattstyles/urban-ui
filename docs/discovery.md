# Discovery

Entrypoint for discovery records: pre-plan captures — triages, surveys, option maps — that exist *before* a plan is worth writing. Where a knowledge record ([[knowledge]]) describes how things are and a plan describes how we'll build, a discovery record holds structured raw material: it is expected to be refined into a plan (and superseded by it) or archived when the moment passes.

## Format

- **Frontmatter** — `tags`, including `discovery`
- State the blockers and open questions explicitly — a discovery record's job is to make the *next* decision cheap
- Link related records with `[[wiki-links]]`

Records live in `docs/discovery/`.
